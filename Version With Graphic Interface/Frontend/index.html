Search for strongly-connected components

How to use:

This app searches for strongly-connected
components in directed graph using
Kosaraju's algorithm

Click here for to gain more information
about the app

Controls:

  1. Click "Vertex" to start setting the vertices
  2. Click "Edge" to interconnect vertices
  3. Click "Reload" if you have done the mistake
  4. Click "Run" to launching the program
